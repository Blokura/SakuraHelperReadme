## 什么是樱花助手?

樱花助手是一款针对哔哩哔哩直播挂机的助手软件。其作用为24小时自动完成所设定的任务，节省时间。

其基本功能如下：

1. 每日签到
2. 直播全区抽奖捡辣条\(小电视、摩天大楼等\)
3. 舰队上任奖励领取\(总督、提督、舰长\)
4. 动态抽奖自动转发
5. 直播间宝箱抽奖
6. 每日银瓜子宝箱领取
7. 风纪委员会自动投票
8. 节奏风暴领取
9. 天选时刻抽奖
10. 主站各类任务

由于篇幅设置，还有很多功能未列举，敬请自行下载体验。
